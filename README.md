# PaperEdge

PaperEdge is a paper trading dashboard for manually tracking sports betting opportunities before risking real money.

The app is designed to help log, review, and analyze simulated arbitrage and betting trades. It does not connect to sportsbooks, place wagers, scrape accounts, bypass geolocation, or guarantee profit.

## Purpose

PaperEdge helps users:

- Record paper trades before games start
- Track expected profit and exposure
- Compare book odds manually
- Review whether a trade would have made money after the event settles
- Track mistakes and lessons learned
- Build better discipline before using real money

## Core Features

- Dashboard overview
- Sportsbook tracking
- New paper trade workflow
- Arbitrage calculator preview
- Trade journal
- Mistakes log
- OddsJam import area
- Manual result review after games settle

## Important Disclaimer

This project is for paper trading and educational tracking only.

PaperEdge does not:

- Place bets
- Connect to sportsbook accounts
- Automate wagering
- Scrape sportsbook balances
- Bypass geolocation
- Guarantee profit
- Provide financial or gambling advice

Always verify odds, sportsbook rules, limits, and market details manually before making any real-world decision.
